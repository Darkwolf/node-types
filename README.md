# Types
## Install
`npm i --save @darkwolf/types`
## Usage
```javascript
// ECMAScript
import { isObject }, types from '@darkwolf/types'
// CommonJS
const types = require('@darkwolf/types')
const {
  isObject
} = types

isObject(Object) // => false
isObject(new Object) // => true
```
